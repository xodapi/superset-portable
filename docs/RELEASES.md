# История версий Superset Portable

## v6.0-rzd (2026-02-07)
**Ключевые изменения:**
- **Новый инструмент обновления**: Добавлен `create_dashboard.exe` (Rust) для обновления данных из Excel/CSV без необходимости установки Python.
- **Производительность**: 
    - Включен **WAL-режим** для SQLite (ускорение записи/чтения).
    - Увеличен пул соединений (`SQLALCHEMY_POOL_SIZE = 10`).
    - Реализован **авто-прогрев** сервера при запуске лаунчера.
- **Размер**: Оптимизирована сборка, удалены лишние файлы документации и тестов Python (~250 MB итоговый размер).
- **Документация**: Обновлены инструкции `HOWTO_UPDATE_DATA.md` и переведены на русский язык.
- **Дашборды**: Исправлен алгоритм обновления данных (теперь не затирает стандартные примеры Superset).
