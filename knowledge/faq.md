# Часто задаваемые вопросы ❓

Ответы на популярные вопросы о портативном Apache Superset.

---

## Общие вопросы

### Что такое Apache Superset?

**Apache Superset** — это современная платформа для исследования и визуализации данных с открытым исходным кодом. Позволяет создавать интерактивные дашборды без программирования.

### Что такое "портативная версия"?

Это специально подготовленная сборка Superset, которая:
- ✅ Не требует установки
- ✅ Работает с USB-накопителя
- ✅ Не нужны права администратора
- ✅ Полностью автономна (оффлайн после установки)

### Можно ли использовать коммерчески?

**Да!** Apache Superset распространяется под лицензией Apache 2.0, которая разрешает:
- Коммерческое использование
- Модификацию и распространение
- Продажу услуг на базе этого ПО

Подробнее: [[Лицензирование]]

## Запуск и работа

### Как запустить Superset?

Дважды кликните `start_superset.bat` или запустите `superset-launcher.exe`.

### Какие логин и пароль по умолчанию?

| | |
|---|---|
| **Логин** | `admin` |
| **Пароль** | `admin` |

> [!WARNING]
> Рекомендуется сменить пароль после первого входа!

### Как сменить пароль?

1. Войдите в систему
2. **Settings → List Users**
3. Найдите пользователя `admin`
4. Нажмите редактирование ✏️
5. Введите новый пароль

### Как остановить Superset?

- Закройте окно консоли
- Или нажмите `Ctrl+C` в консоли
- Или используйте иконку в системном трее

## Проблемы

### Superset не запускается

Проверьте:
1. Порт 8088 свободен: `netstat -an | findstr 8088`
2. Python установлен: `python\python.exe --version`
3. Нет кириллицы в пути к папке

Подробнее: [[Ошибки запуска]]

### Ошибка "Fatal error in launcher"

Обычно связана с кириллицей в пути. Переместите папку в путь без русских букв:

❌ `C:\Пользователи\Иван\superset-portable\`  
✅ `C:\tools\superset-portable\`

Подробнее: [[Кириллица в путях]]

### Запрос выполняется слишком долго

1. Добавьте `LIMIT` в запрос
2. Создайте индексы в базе данных
3. Увеличьте таймаут в конфигурации:

```python
# superset_config.py
SUPERSET_WEBSERVER_TIMEOUT = 120
```

### Не отображаются русские названия столбцов

Проверьте кодировку базы данных (должна быть UTF-8).

Для PostgreSQL:
```sql
SET client_encoding TO 'UTF8';
```

## SQL Lab

### Как сохранить запрос?

1. Напишите запрос в SQL Lab
2. Нажмите **Save** или `Ctrl+S`
3. Введите название и описание

### Как создать датасет из запроса?

1. Выполните запрос
2. **Save → Save Dataset**
3. Теперь можно строить диаграммы!

## Дашборды

### Как добавить фильтр на дашборд?

1. Откройте дашборд в режиме редактирования
2. **Components → Filter Box**
3. Перетащите на дашборд
4. Настройте поля фильтрации

Подробнее: [[Фильтры]]

### Как экспортировать дашборд?

- **В PNG**: Меню дашборда → Download as Image
- **В YAML**: Settings → Export Dashboards
